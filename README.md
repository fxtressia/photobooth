# A Photobooth Project

The flow is as follows:
- The user logs into the machine for a set period of time, which by default is 10 minutes, excluding the time it takes for the system to wait for the camera's shutter to complete. Inside the machine, they can open two apps.
- The camera app, which will
 take pictures
- The pictures-editing app, which has a user interface similar to Canva, where they could pick templates, add their pictures along with available stickers, and order prints of their photobooth pictures.
- After leaving the machine, they can log again into the pictures-editing app. 

Ordering prints will remain available so long as the're an active machine at the moment. Users will have to physically pick the picture the moment they're printed though, so they're have to be present nearby.
